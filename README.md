
# Meu Currículo Online e Interativo👌💕

Meu site é um currículo online interativo, desenvolvido com apoio de ferramentas de Inteligência Artificial [IA](https://chatgpt.com/) e ideias pensadas para facilitar ao selecionador o acesso rápido às minhas informações profissionais. O objetivo é apresentar dados como experiência, formação, competências e contatos de forma organizada, moderna e fácil de navegar, permitindo que recrutadores visualizem meu perfil de maneira prática e profissional. O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, e está hospedado na plataforma GitHub por meio do serviço GitHub Pages.
